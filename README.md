# 🚀 FinScoreTech

A smart, interactive, multi-page website that analyzes your **financial health** in seconds.  
FinScoreTech blends stunning UI, AI chatbot, animations, charts, and a personalized experience for users of all ages — even kids! 🧠💰✨

---

## 🌟 Features

- 🔢 FinScore Calculator (Income vs Expenses vs Savings)
- 📊 Interactive Pie Chart for Expense Breakdown
- 💬 AI Chatbot Assistant (Ask financial questions!)
- 🎯 Smart Tips & Recommendations based on FinScore
- 📱 Mobile Responsive UI
- 🛡️ Planning to Add: Secure User Login & History Tracking

---

## 💻 Tech Stack

- HTML, CSS, JavaScript
- Chart.js
- OpenAI GPT (via JavaScript integration)
- GitHub Pages Hosting
